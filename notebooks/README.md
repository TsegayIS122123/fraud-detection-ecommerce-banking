# Notebooks Directory

This directory contains all Jupyter notebooks for the fraud detection project.

## Notebook Structure

1. **eda-fraud-data.ipynb** - EDA for e-commerce fraud data
2. **eda-creditcard.ipynb** - EDA for credit card fraud data  
3. **feature-engineering.ipynb** - Feature engineering for both datasets
4. **modeling.ipynb** - Model building and evaluation
5. **shap-explainability.ipynb** - Model interpretability using SHAP
